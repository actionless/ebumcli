# ebumcli

(EBU Meter CLI / EBU Mini CLI) - extremely minimalistic EBU R 128 LUFS and TruePeak meter CLI (using libebur128).

Tested on Linux and MacOS (Homebrew).


## Installation

### Arch Linux

```console
$ pikaur -S ebumcli-git
$ ebumcli --help
```

### Other

```console
$ make
$ ./ebumcli --help
```
